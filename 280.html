Projects - EECS 280

Natural Language Processing Content Classifier (Piazza)

Built program that employs the Multinomial Naïve Bayes classifier probabilistic model to predict the topics of online forum posts, correctly labeling 87% of 2,988 real posts Architected and developed program in C++ using maps and binary search trees Learned about various machine learning models, choosing the most efficient solution for the classification task


Computer Vision

Implemented content-aware image scaling by computing the cost of pixels in a matrix. The program outputs another image that will have a reduced amount of redundant pixels.


Web Backend

Recreated a virtual office-hours queue for EECS 280 with the use of linked lists, iterators, and JSON objects. As the goal of this project was to introduce students to working with APIs, this program was able to be hosted on a web server upon completion.


Euche Simulation

Simulated euchre, a card game popular in the state of Michigan. Key components of this project included implementing algorithms for shuffling/dealing cards, defining card values in relation to one another, and creating different “players” for the user to compete against by using polymorphism/derived classes.
